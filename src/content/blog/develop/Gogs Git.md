## Gogs安装  
1. 下载Gogs  
地址https://github.com/gogs/gogs/releases  
2. 将gogs的压缩包解压C:\gogs  
3. 配置环境变量
4. cmd输入gogs web
5. 浏览器http://localhost:3000

## NSSM 安装GIT服务
1. 下载nssm  
地址:https://nssm.cc/download  
2. cmd: nssm install gogsgit  
3. 选择gogs.exe  
4. 参数web
