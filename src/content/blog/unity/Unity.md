## 音频转换格式

https://github.com/BtbN/FFmpeg-Builds

~~~bash
ffmpeg -i input.mp3 output.wav      # MP3 → WAV
ffmpeg -i input.aac output.flac     # AAC → FLAC
ffmpeg -i input.m4a output.ogg      # M4A → OGG
~~~
